# Capacita+

## Descrição
Um portal livre de curadoria para cursos de capacitação.

Versão estática acessível em: [https://heberfhlemes.github.io/capacita-mais](https://heberfhlemes.github.io/capacita-mais)


## Testando localmente

É possível rodar localmente com `npm` ou `Docker`:

#### npm
```bash
npx serve public
```
Acesse em: http://localhost:3000

#### Docker (Nginx)
```bash
docker compose up -d
```
Acesse em: http://localhost:8080
